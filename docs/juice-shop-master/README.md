## Project Description
This repository documents several hacking challenges solved within the OWASP Juice Shop environment.
Each challenge includes a written report, related materials, and a demonstration video showing how the vulnerability is exploited.

- A technical description
- Exact exploitation steps
- Risks & impact
- Results
- A demonstration video

## Quickstart

### 1. Install Dependencies
```bash
cd docs-site
npm install
```

### 2. Start the Application
```bash
npm start
```

### 3. Visit

Now visit the application at:
```
http://localhost:3001
```

## Prerequisites

Before running the project, ensure you have:

- Node.js

- npm

- A modern web browser

- Basic understanding of web security concepts

## Usage

1. Start the Docusaurus documentation server.

2. Navigate to **Juice Shop Master** in the sidebar.

3. Select a challenge (XSS, CAPTCH or Admin).

4. Follow the documented steps and watch the attached video.
