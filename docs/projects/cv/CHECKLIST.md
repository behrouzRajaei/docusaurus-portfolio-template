# Project Checklist

# Checkliste für Deinen CV

#  **Der Self-Check** 

Den größten Lernerfolg hast Du, wenn Du Dir wirklich Gedanken über die einzelnen Punkte aus dem Leitfaden, den Videos und aus den Calls gemacht hast\! 

Jetzt ist es erst mal an der Zeit, Dein Werk selber noch mal zu überprüfen und Dich selber zu reflektieren.

Wenn Du den entsprechenden Punkt **verstanden** **und** in Deinem CV **umgesetzt** **hast, aktiviere bitte das entsprechende Kontrollfeld** ✅

Bitte sende Deinen CV (als Link zu deinem Google Docs) **erst dann zur Nachprüfung**, wenn Du ALLE Punkte gewissenhaft bearbeitet hast. Falls noch Fragen offen sind, kannst Du sie direkt in den Live-Calls stellen.

| Rubrik | Unterpunkt /Beschreibung | erledigt |
| ----- | ----- | ----- |
|  |  |  |
| **Bewerbungsfoto** | **Fotoqualität und Hintergrund** |  |
| Auflösung und Format | Ist das Foto hochauflösend (mindestens 300 dpi) und das Format passend für eine Integration in den Lebenslauf (idealerweise 4,5 x 6 cm) |  |
| Hintergrund | Ist der Hintergrund neutral und minimalistisch? Ein einfacher, einfarbiger Hintergrund (z. B. hellgrau oder weiß) ist in der IT-Branche üblich. |  |
| Lichtverhältnisse | Ist die Beleuchtung gleichmäßig, sodass das Gesicht gut zu erkennen ist? Vermeide harte Schatten oder schlechte Ausleuchtung, die Unprofessionalität suggerieren könnten. |  |
|  | **Branchenübliche Kleidung** | **✓** |
| Kleidungs-Stil | Entspricht die Kleidung einem smart-casual Dresscode und wirkt gepflegt und branchentypisch? |  |
| Farbwahl | Unterstützen die Farben der Kleidung ein freundliches und authentisches Erscheinungsbild, ohne abzulenken? Farben wie Blau, Weiß oder Grautöne wirken professionell, auffällige Farben sollten vermieden werden. |  |
|  | **Gesichtsausdruck und Blickrichtung** | **✓** |
| Augen | Sind die Augen wach und aufmerksam? Insbesondere bei Entwicklern wird erwartet, dass sie aufmerksam und konzentriert wirken. Müde oder verschlossene Blicke könnten als fehlende Motivation interpretiert werden. |  |
| Haltung | Ist die Körperhaltung entspannt, aber dennoch aufrecht? |  |
| Bildausschnitt | Ist der Bildausschnitt auf Kopf und Schultern fokussiert?  |  |
|  | **Frisur** | **✓** |
| Gepflegtes Aussehen | Sind die Haare (+ggf. Bart) ordentlich gestylt, ohne exzentrisch zu wirken? |  |
| Passend zur Position | Wirkt die Frisur modern und dennoch zur Branche passend? Stark extravagante Frisuren könnten bei einem Junior-Webentwickler als unstimmig interpretiert werden. |  |
|  | **Schmuck** | **✓** |
| Schmuck und Accessoires | Ist der Schmuck minimalistisch? Auffälliger Schmuck wirkt ablenkend.  |  |
|  | **Allgemeiner Eindruck** | **✓** |
| Authentische Darstellung | Geht der Blick direkt in die Kamera und wirkt der Gesichtsausdruck freundlich und authentisch? |  |
| Teamfähigkeit signalisieren | Wirkt das Foto einladend und aufgeschlossen? Da die Zusammenarbeit im Team eine zentrale Rolle spielt, sollte man nicht übermäßig ernst oder unnahbar wirken. |  |

| Rubrik | Unterpunkt /Beschreibung | erledigt |
| ----- | ----- | ----- |
|  |  |  |
| **Kontakt** | **Kontaktinformationen** | **✓** |
| Namensangaben | Wurde sowohl Vor- als auch der Nachname genannt? Außer bei Doppelnamen wurde auf die Nennung weiterer Vornamen verzichtet. "So werde ich gerufen\!" |  |
| Berufsbezeichnung (Zeile 1\) | Hier ist in erster Zeile die Berufsbezeichnung eingetragen, auf die du dich bewirbst. Achte auf die Durchgängigkeit der Schreibweise im gesamten Dokument (Beispiel Frontend Developer | Front-End Developer) |  |
| Berufsbezeichnung ggf. Titel (Zeile 2\) | Hast du bereits eine für die Bewerbung absolut relevante Berufsbezeichnung oder einen akademischen Titel, kann dieser in einer zweiten Zeile hinzugefügt werden. Diese sollte allerdings zwei Schriftgrößen kleiner sein als in Zeile 1, um Irritationen zu vermeiden. |  |
| Postadresse | Ist die Postadresse vollständig und korrekt angegeben, sodass du erreichbar bist? |  |
| Telefonnummer | Ist eine aktuelle Telefonnummer angegeben, unter der du gut erreichbar bist? Zusätzlich sollte überprüft werden, ob die Schreibweise der Telefonnummer der DIN-Norm 5008 entspricht. (Beispiel: \+49 176 12345678\) Die Nummer ist zudem als Link angelegt und ermöglicht ein direktes Anrufen. |  |
| E-Mail-Adresse | Ist die E-Mail-Adresse seriös und professionell? Es sollte eine Adresse in Bezug auf die eigene Portfolio-Domain verwendet werden, um die Affinität zum eigenen Hosting zu beweisen (z. B. [hallo@meine-domain.com](mailto:hallo@meine-domain.com)). Der Link sollte klickbar sein und direkt zum Mailprogramm leiten (mailto:) |  |
| Portfolio-Link | Ist ein Link zum Portfolio angegeben, der Einblick in deine relevanten Projekte und Arbeiten gibt? Der Link sollte unbedingt auf https:// weiterleiten, da Browser unverschlüsselte http:// Links nicht mehr zuverlässig öffnen. Richtiges Beipiel: [https://mein-portfolio.de](https://mein-portfolio.de/)  In der Darstellung eines Hyperlinks im Text eines Lebenslaufs kann in den meisten Fällen auf „https://“ und „www.“ verzichtet werden, ohne die Professionalität oder die Funktionalität zu beeinträchtigen. |  |
| Github und LinkedIn Profil | Ist dein Github und LinkedIn Profil (falls vorhanden) verlinkt?  |  |

| **About Me** | **Grundlegende Struktur und Präsentation** | **✓** |
| Formatierung und Rechtschreibung | Wurde der Lebenslauf auf Formatierungs- und Rechtschreibfehler überprüft, um einen professionellen Eindruck zu hinterlassen? |  |
| Inhaltliche Texte | Ist der Lebenslauf klar strukturiert und prägnant formuliert, ohne lange, uninteressante Texte? Wichtige Informationen sollten in einem übersichtlichen Format präsentiert werden. |  |
| Richtigkeit der Angaben | Sind die gemachten Angaben im Lebenslauf wahrheitsgemäß und überprüfbar, um das Vertrauen des potenziellen Arbeitgebers zu gewinnen? |  |
|  | **Selbstpräsentation und Kompetenzdarstellung** | **✓** |
| Selbstbeschreibung | Ist deine Selbstbeschreibung klar, präzise und werden messbare Erfolge genannt, um deine Kompetenzen zu verdeutlichen? |  |
| Klare Kompetenzen | Ist die Darstellung der technischen Fähigkeiten präzise und klar? Es sollten konkrete Beispiele genannt werden, um deine Fachkenntnisse deutlich zu machen. |  |
| Ausgeglichenes Selbstbewusstsein | Wirkt die Darstellung im Lebenslauf ausgeglichen und authentisch? Ein Lebenslauf sollte weder ein zu geringes Selbstbewusstsein, das auf Unsicherheit hinweist, noch ein übertriebenes Selbstbewusstsein zeigen, das als angeberisch oder unglaubwürdig wahrgenommen werden könnte. Du solltest deine Stärken klar, aber ohne Übertreibung darstellen und deine Schwächen nicht übermäßig betonen. |  |
|  | **Relevanz der Inhalte und Fokus** | **✓** |
| Fokus auf dich als Fachkraft nicht als Lernender | Liegt der Schwerpunkt der Darstellung auf praktischen Anwendungen und Projekten, die zeigen, wie du das Gelernte aktiv in der Praxis eingesetzt hast, statt den Lernprozess überzubetonen? |  |
| Motivation für Quereinstieg | Wird im „Über mich“-Abschnitt angegeben, warum du dich für den Quereinstieg entschieden hast und was dich motiviert? |  |
| Sprechweise in der Gegenwart | Wird im Lebenslauf klar dargestellt, wie du deine Kenntnisse aktuell einsetzt und welche Ergebnisse daraus resultieren, statt sich nur auf die Vergangenheit zu beziehen? |  |
| Aktuelle Tätigkeiten und Erfolgen | Der Fokus liegt auf den neuesten und relevantesten Erfolgen, die für die angestrebte Position von Bedeutung sind. Nicht: bisherige berufliche Laufbahn zu dominant in den Vordergrund stellen  |  |
| Ausblick in die Zukunft | Enthält der Abschnitt einen Ausblick, wie du dich in der neuen Branche weiterentwickeln möchtest? |  |
|  | **Fähigkeiten und Soft Skills** |  |
| Konkrete Soft Skills | Werden im Lebenslauf Soft Skills genannt, die über allgemeine, selbstverständliche Fähigkeiten wie Pünktlichkeit oder Zuverlässigkeit hinausgehen? Soft Skills wie Teamarbeit oder Kommunikationsfähigkeit sollten durch konkrete Beispiele belegt werden. |  |
| Fokus auf das Handeln statt auf das Wollen | Werden konkrete Beispiele genannt, wie du deine Fähigkeiten bereits eingesetzt hast? (aktive Haltung) |  |
| Richtiges Maß an Lernbedarf | Zeigt der Lebenslauf deine Bereitschaft, Verantwortung zu übernehmen und produktiv zu arbeiten, statt den Lernbedarf übermäßig zu betonen? |  |
